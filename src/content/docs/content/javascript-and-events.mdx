Making an interactive web page requires a host of _events_: did users change the status of an element
(e.g. `checkbox`) or just click it (e.g. a `button`/`div`)? Javascript components "listen" for events
to happen and we can bind function to those events to trigger additional functionality.

Consider the following `JSX`:
```javascript
import React, {useState} from 'react';

const EventsButton = () => {
  const handleClick = (evt) => {
    alert("Button was clicked!");
    console.log("Button was clicked!", evt);
  };

  return (
    // 2. Pass the function reference to onClick
    <button onClick={handleClick}>
      Click Me
    </button>
  );
}

export default EventsButton;
```
